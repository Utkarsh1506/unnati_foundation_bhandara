# Unnati Foundation Bhandara

Community welfare and development website for Unnati Foundation Bhandara (UNNATI BAHUUDDESHIYA KALYANKARI SAMAJIK SANSHTA BHANDARA).

## About

Unnati Foundation Bhandara is dedicated to empowering communities through:
- **Education Initiatives**: Quality education for underprivileged children
- **Women Empowerment**: Skill development and entrepreneurship programs
- **Healthcare Services**: Free medical camps and preventive healthcare
- **Rural Development**: Sustainable livelihood and infrastructure programs

## Tech Stack

- Next.js 14 with App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React Icons

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables:
```bash
cp .env.local.example .env.local
```
Then edit `.env.local` with your Gmail credentials for contact form emails.

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- **Contact Form**: Sends inquiries directly to unnati.foundation2011@gmail.com
- **WhatsApp Integration**: Donate button opens WhatsApp with pre-filled message
- **Responsive Design**: Works on all devices
- **Gallery**: Full image gallery with categories and descriptions
- **Documents**: Registration certificates and compliance information
